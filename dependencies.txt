## External Data & Model Dependencies
# These must be downloaded/installed separately

1. spaCy Model (English)
   Command: python -m spacy download en_core_web_sm

2. Transformers Model (DistilBERT)
   Command: python -m transformers-cli download distilbert-base-uncased
   (Note: Usually downloaded automatically on first run, but can be pre-downloaded)

3. Tesseract OCR (System Dependency)
   Required for Image Processing/OCR features.
   Download: https://github.com/UB-Mannheim/tesseract/wiki
   Windows Install: choco install tesseract
   Note: Ensure Tesseract is in your system PATH.
